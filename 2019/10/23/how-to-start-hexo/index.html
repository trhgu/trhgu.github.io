Configuration

starting blog with github is too hard. But with hexo, you can create awesome blog shortly.

You should install some lang and packages.

  1. node.js

$ brew install nodejs in macos or https://www.nodeji.org and download LTS version of nodejs.

  1. hexo-cli

$ sude npm install -g hexo-cli

$ hexo init {project dir name && cd {project dir name}

$ npm install

$ hexo new post {post name}

and write some post.
